Knockout Games

These games are made to be played in a browser and on mobile devices. Cost will be low to none as we simply build games for fun and to showcase what we can do as developers. Game development is actually a good test of programming best practices. The development of a game requires a very well organized code base and is quite suited to object orientated programming. From abstracting your game objects, controlling states, and building game physics, game design can be complex. Staying organized while using best practices is essential to not being overwhelmed by the scope a game can grow to.
