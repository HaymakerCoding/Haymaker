Customer Focused Applications

City Match Play Championship

City Match Play Championship is one of ClubEG's many tournaments. This application is more than it seems! It is actually an Angular built SPA that works as a template and content management system for the company. We built it as single app that can be the point of entry for any number of tournament domains they setup. This allows them to then use an accompanying admin application to supply content specific to each tournament. Allowing them to upload and crop pictures, supply content for sections and manager registrations. In this way they can build out as many tournaments as they would like using just one application that handles the formating and presentation as well as functions for login and registering.

ClubEG

ClubEG is a fantastic membership based golfing service, offering social and competitive events for their members, as well as discounts at many courses.

This golfing web site has functionality for members to login and register for upcomming events and trips. Built using a traditional combination of PHP for the server side and plain JavaScript/JQuery for the client and Bootstrap 4 for responsive design and style. A key to this site was adding content management so administrators can add pages themselves with little programming knowledge. This is demostrated with a custom dynamic course page template that can be used to make an number of beautiful course pages.

The Slammer Tour

The Slammer Tour is a popular golf league with many members, offering great play for all competition preferences. There is a lot going on with this site, as members can register for events, view stats, profiles and results and coordinators can manage events and scoring.

A key part of this project was a complex scoring system. Many algorithums are used to calculate different scenarios for players to earn points, based on their performance in their group, as well as overall. This site is an overhaul of an older version. We were able to successfully update many of the features to be mobile responsive as well as add new features such as scoring and image uploading. Included in the refresh was updating the database services and PHP version.

CourseDB

This site is mainly for course manager's of various golf courses to supply information about their course and create course listings using our custom content management system. Courses in the database can be assigned to their respective manager's, who can then create a basic, standard or premium listing.

This site is built using the modern JavaScript framework Angular, consuming data from a custom build PHP api. This allows clear seperation between the client side and the server side and allows reusing the api end points anywhere. Keys to this development are the token based authentication system as well as a great UI for setting up courses. The UI allows secure uploading of data and pictures, with a very sleek cropping tool for the images so they conform to the template. Also included is a review system for visitors.
